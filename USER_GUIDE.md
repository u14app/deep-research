# 🎯 用户指南：如何使用双模式功能

## 📍 现在您的界面上有什么？

当您访问网站时，在**研究主题**区域的右上角，您会看到一个**模式选择器**：

```
┌─────────────────────────────────────────────────────┐
│ 1. 研究方向                [🔍 普通模式 ▼]  [➕]     │
├─────────────────────────────────────────────────────┤
│                                                     │
│  根据选择的模式，这里会显示不同的输入表单            │
│                                                     │
└─────────────────────────────────────────────────────┘
```

---

## 🔍 普通模式（默认）

**适用场景**：任何主题的深度研究

### 界面展示
```
┌─────────────────────────────────────────────────────┐
│ 1. 研究方向                [🔍 普通模式 ▼]  [➕]     │
├─────────────────────────────────────────────────────┤
│                                                     │
│ 1.1 研究主题                                        │
│ ┌─────────────────────────────────────────────┐   │
│ │ 任何您想了解的问题...                        │   │
│ │                                             │   │
│ └─────────────────────────────────────────────┘   │
│                                                     │
│ 1.2 本地研究资源（可选）                            │
│ [➕ 添加资源]                                       │
│                                                     │
│ [开始思考]                                          │
└─────────────────────────────────────────────────────┘
```

### 使用步骤
1. 确保模式选择器显示 "🔍 普通模式"
2. 在文本框中输入您想研究的任何主题
3. （可选）添加本地资源文件
4. 点击"开始思考"按钮

### 示例输入
```
输入主题: "2025年人工智能发展趋势"
输入主题: "区块链技术在金融领域的应用"
输入主题: "可持续能源的未来发展"
```

## 🧬 专业模式（基因研究）

**适用场景**：基因功能研究、蛋白质分析、分子生物学

### 界面展示
```
┌─────────────────────────────────────────────────────┐
│ 1. 研究方向            [🧬 专业模式 ▼]  [➕]         │
├─────────────────────────────────────────────────────┤
│                                                     │
│ Gene Symbol                                         │
│ ┌─────────────────────────────────────────────┐   │
│ │ e.g., TP53, BRCA1, lysC                     │   │
│ └─────────────────────────────────────────────┘   │
│                                                     │
│ Organism                                            │
│ ┌─────────────────────────────────────────────┐   │
│ │ e.g., Homo sapiens, E. coli                 │   │
│ └─────────────────────────────────────────────┘   │
│                                                     │
│ Research Focus                                      │
│ ☑ General Function    ☑ Disease Association        │
│ ☐ Protein Structure   ☐ Expression Analysis        │
│ ☐ Protein Interactions ☐ Evolutionary Analysis     │
│ ☐ Therapeutic Potential                             │
│                                                     │
│ Disease Context (Optional)                          │
│ ┌─────────────────────────────────────────────┐   │
│ │ e.g., breast cancer, Alzheimer's disease    │   │
│ └─────────────────────────────────────────────┘   │
│                                                     │
│ [🧬 Start Gene Research]                           │
└─────────────────────────────────────────────────────┘
```

### 使用步骤
1. 点击模式选择器，选择 "🧬 专业模式（基因研究）"
2. 界面会自动切换到基因研究表单
3. 填写基因信息：
   - **Gene Symbol**: 输入基因符号（如 TP53, BRCA1）
   - **Organism**: 输入物种名称（如 Homo sapiens）
   - **Research Focus**: 选择一个或多个研究焦点
   - **Disease Context**: （可选）输入相关疾病
4. 点击 "🧬 Start Gene Research" 开始研究

### 研究焦点说明

| 焦点 | 中文名称 | 研究内容 |
|------|---------|---------|
| **General Function** | 常规功能 | 基因的基本功能和作用 |
| **Disease Association** | 疾病关联 | 基因与疾病的关系 |
| **Protein Structure** | 蛋白质结构 | 蛋白质的三维结构 |
| **Expression Analysis** | 表达分析 | 基因表达模式 |
| **Protein Interactions** | 蛋白质相互作用 | 蛋白质之间的相互作用 |
| **Evolutionary Analysis** | 进化分析 | 基因的进化历史 |
| **Therapeutic Potential** | 治疗潜力 | 基因的治疗应用 |

### 示例输入

#### 示例 1：人类肿瘤抑制基因
```
Gene Symbol: TP53
Organism: Homo sapiens
Research Focus: ☑ General Function, ☑ Disease Association, ☑ Therapeutic Potential
Disease Context: cancer
```

#### 示例 2：大肠杆菌代谢基因
```
Gene Symbol: lysC
Organism: E. coli
Research Focus: ☑ General Function, ☑ Protein Structure
Disease Context: (留空)
```

#### 示例 3：乳腺癌相关基因
```
Gene Symbol: BRCA1
Organism: Homo sapiens
Research Focus: ☑ Disease Association, ☑ Protein Interactions
Disease Context: breast cancer
```

## 🔄 如何切换模式？

### 方法：点击下拉选择器

1. 在页面顶部找到模式选择器
2. 点击下拉箭头 ▼
3. 从菜单中选择：
   - **🔍 普通模式** - 切换到通用研究
   - **🧬 专业模式（基因研究）** - 切换到基因研究

### 切换效果

```
普通模式 → 专业模式
┌─────────────────────┐      ┌─────────────────────┐
│ 文本框输入           │  →   │ 基因符号            │
│ 任意主题            │      │ 物种                │
│                     │      │ 研究焦点            │
└─────────────────────┘      └─────────────────────┘

专业模式 → 普通模式
┌─────────────────────┐      ┌─────────────────────┐
│ 基因符号            │  →   │ 文本框输入           │
│ 物种                │      │ 任意主题            │
│ 研究焦点            │      │                     │
└─────────────────────┘      └─────────────────────┘
```

## 🆘 常见问题

### Q: 我看不到模式选择器怎么办？
A: 模式选择器位于页面顶部 "1. 研究方向" 区域的右上角。如果看不到，请尝试刷新页面。

### Q: 切换模式后，之前的输入会消失吗？
A: 是的，切换模式会清空输入框。建议在切换前保存重要内容。

### Q: 专业模式需要特殊的API密钥吗？
A: 不需要。专业模式使用相同的AI API配置，只是研究流程和数据源不同。

### Q: 可以同时使用两种模式吗？
A: 不可以。同一时间只能使用一种模式。但您可以随时切换。

### Q: 专业模式的研究结果格式有什么不同？
A: 专业模式会生成包含基因功能、蛋白质信息、表达数据等专业内容的结构化报告。
